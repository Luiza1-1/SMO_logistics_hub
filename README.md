# SMO_logistics_hub
## Курсовая работа по "Архитектуре программных систем" 

Реализован будет логистический складской хаб продовольственных товаров. А точнее рассмотрим поступление фур на слад продовольственных товаров и дальнейшее их распределение туда.

### Sequence_diagram

ИЗ1 - Пуассоновский поток: случайное время прибытия грузовиков

ПЗ2 - Равномерное распределение: время разгрузки колеблется в диапазоне

Д10З3 - Буферизация на свободное место: заявка ставится на первое свободное место

Д10ОЗ - Самая старая в буфере: проверка дедлайна у самой старой заявки при переполнении

Д2П1 - Приоритет по номеру прибора: П1 для скоропортящихся, П2 для обычных

Д2Б2 - LIFO: последняя пришедшая заявка обслуживается первой
