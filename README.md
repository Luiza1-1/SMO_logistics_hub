## 2.Первичные артефакты
### Sequence_diagram

o Источник (И) - Поток грузовов с товаром, подъезжающий к складу

o Буфер (БП) - Парковочные места для ожидания разгрузки на
территории склада

o Прибор (П) - Разгрузочный пост (ворота) с бригадой грузчиков

o Диспетчер (Д) - Складской диспетчер, управляющий потоками машин.

o Заявка (требование) - Грузовик, нуждающийся в разгрузке, у которого есть
свой приоритет.

Технические требования:

o ИЗ1 - Пуассоновский поток: случайное время прибытия грузовиков

o ПЗ2 - Равномерное распределение: время разгрузки колеблется в диапазоне

o Д10З3 - Буферизация на свободное место: заявка ставится на первое свободное место

o Д10ОЗ - Самая старая в буфере: проверка дедлайна у самой старой заявки при переполнении

o Д2П1 - Приоритет по номеру прибора: П1 для скоропортящихся, П2 для обычных

o Д2Б2 - LIFO: последняя пришедшая заявка обслуживается первой
